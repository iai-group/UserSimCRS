nlg_generative_lm
=================

.. py:module:: nlg_generative_lm

.. autoapi-nested-parse::

   Generative natural language generation using a language model.

   Note that the language model is expected to be hosted externally on a ollama
   server. Also, the prompt used to generate utterances is expected to have the
   following placeholders:
   - "dialogue_acts": to be replaced by the stringified dialogue acts.
   - "annotations": to be replaced by the stringified annotations (if provided).
