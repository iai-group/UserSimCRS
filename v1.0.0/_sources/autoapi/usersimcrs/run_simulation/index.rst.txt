.. py:function:: main(config: confuse.Configuration, agent: dialoguekit.participant.agent.Agent) -> None

   Executes the specified configuration.

   Loads domain and interaction model. Initializes agent and user. Runs the
   simulation.

   :param config: Configuration generated from YAML configuration file.
   :param agent: Conversational agent.

   :raises TypeError: if the agent is not an instance of Agent.


.. py:function:: simulate_conversation(agent: dialoguekit.participant.agent.Agent, user_simulator: usersimcrs.simulator.user_simulator.UserSimulator) -> dialoguekit.core.dialogue.Dialogue

   Simulates a single conversation and returns the resulting dialogue.

   :param agent: An agent.
   :param user_simulator: A user simulator.

   :returns: The simulated dialogue.


.. py:function:: parse_args() -> argparse.Namespace

.. py:function:: load_config(args: argparse.Namespace) -> confuse.Configuration

   Loads config from config file and command line parameters.

   Loads default values from `config/default/config_default.yaml`. Values are
   then updated with any value specified in the command line arguments.

   :param args: Arguments parsed with argparse.


.. py:function:: get_NLU(config: confuse.Configuration) -> dialoguekit.nlu.intent_classifier.IntentClassifier

   Returns an NLU component.

   Only supports DialogueKit intent classifiers.

   :param config: Configuration for the simulation.

   :raises ValueError: Unsupported intent classifier.

   :returns: An NLU component.
